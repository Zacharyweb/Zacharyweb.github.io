express学习笔记(1)

发表于 2017-02-07 | 分类于 后端开发 | 点击量 次

Express是用来加快我们开发Web应用程序的一个Web开发框架,在此记录一下express框架的基础使用。

基本使用

  • 安装npm install express

  • 创建一个app实例

  • 在项目中使用：

基本API

app.all()

app.all(path, callback [, callback …])：对任何请求都会调用该处理函数。

app.get()

app.get(path, callback [, callback …])：该方法可以处理get请求的url。

app.listen()

app.listen(port, [hostname], [backlog], [callback])：和原生的server.listen 是一样的。

app.post()

app.post(path, callback [, callback …])：该方法可以处理post请求的url。

app.route()

app.route(path)：路由。

app.set()

app.set(name, value)

  • views 可以设置模板文件所在的位置
  • view engine 可以设置使用哪个模板引擎
  • view cache 是否缓存模板

app.use()

app.use([path,] function [, function…])

  • 第一个参数 path 可选的，表示要匹配的路径的意思
  • 第二个参数一般是一个函数，函数里面的参数是：req,res,next
  • req表示请求对象
  • res表示响应对象
  • next 表示下一个要匹配的中间件

Request

req.app

我们可以通过给 app 动态的挂载一些属性，然后就可以在req对象上直接通过app访问app的一些属性.

req.body

可以通过req.body 获取post请求提交的表单参数。

注意：如果要使用req.body，必须使用第三方中间件

req.hostname

可以获取主机名，或者说是域名。

req.ip

可以获取客户端请求的IP地址。

req.params

当匹配/user/:uid/:haha到这种url的时候，直接将 uid、hah 挂载到 req.params 对象上。

req.path

获取客户端请求路径部分，不包含查询字符串。

req.query

可以直接通过它获取get请求参数。

Response

res.locals

我们可以通过给locals挂载一些通用的数据，方便在多个地方使用。

res.download()

res.download(path [, filename] [, fn])：可以主动让客户端下载这个文件。

res.json()

res.json([body])：直接向客户端响应一个json格式的字符串。

res.redirect()

res.redirect([status,] path)：让客户端重定向。

  • 例如：让客户端跳转到首页：res.redirect('/')。
  • 让客户端重定向到登录页面：res.redirect('/login')。
  • 让客户端，直接在当前页面刷新：res.redirect('back')。

res.render()

res.render(view [, locals] [, callback])：向客户端响应一个模板，然后注入模板数据。

res.send()

res.send([body])：向客户端发送普通文本，它自己会end。
